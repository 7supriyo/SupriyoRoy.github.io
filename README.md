# Supriyo Roy - Personal Portfolio

Welcome to my personal portfolio website! This is a modern, responsive portfolio built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Dark Theme**: Eye-friendly dark mode with gradient accents
- **Sections Included**:
  - Hero/Landing Section
  - About Me
  - Projects Showcase
  - Skills
  - Contact Form
  - Social Links

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Variables
- **Icons**: Font Awesome 6.4.0
- **Static Site Generator**: Jekyll (for GitHub Pages)

## 📝 Customization

### Update Your Information

1. **Name and Title** - Edit in `index.html`:
   ```html
   <h1 class="name">Supriyo Roy</h1>
   <p class="title">Developer | Creator | Innovator</p>
   ```

2. **About Section** - Update your bio in the About section

3. **Projects** - Modify the project cards with your projects

4. **Skills** - Add your actual skills in the Skills section

5. **Social Links** - Update URLs to your actual social profiles

6. **Contact Email** - Change the email in the Contact section and form

### Color Customization

Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;      /* Change primary color */
    --secondary-color: #8b5cf6;    /* Change secondary color */
    --dark-bg: #0f172a;            /* Change background */
    /* ... other variables ... */
}
```

## 🚀 Deployment

This site is automatically deployed to GitHub Pages at:
**https://7supriyo.github.io/**

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/7supriyo/SupriyoRoy.github.io.git
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Serve locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Open http://localhost:4000 in your browser

## 📧 Contact Form

The contact form uses `mailto:` to send emails. To enable email functionality:

1. Update the email address in `index.html` (look for `your.email@example.com`)
2. Users will click "Send Message" and their default email client will open with the pre-filled content

Alternative: For server-side form handling, consider using services like:
- Formspree
- Basin
- Netlify Forms
